# Python Trivia Game

A simple and interactive command-line trivia game built using Python.  
The game asks random questions, checks user answers, and displays the final score.
This project is beginner-friendly and demonstrates the use of Python basics and the `random` module.

# Features

- Random trivia questions
- User input validation
- Score tracking
- Simple and easy-to-understand logic
- Runs in terminal / command prompt

# Technologies Used

- Python 3
- `random` module
